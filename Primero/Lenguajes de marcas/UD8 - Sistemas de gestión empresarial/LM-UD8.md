# UD8 - Sistemas de gestión empresarial (ERP)
## Lenguajes de marcas

---

# ¿Qué son los ERP?

Son sistemas que permiten gestionar de forma centralizada los recursos y procesos de una empresa.

Un ERP integra distintas áreas:

- Contabilidad
- Recursos humanos
- Ventas
- Compras
- Almacén
- Producción

Todo conectado en una única aplicación.

---

# ¿Qué son los CRM?

Son sistemas orientados a la gestión de clientes y relaciones comerciales.

Permiten:

- Registrar clientes
- Gestionar ventas
- Hacer seguimiento comercial
- Controlar incidencias
- Automatizar marketing

El objetivo principal es mejorar la relación con los clientes.

---

# Características de los ERP y CRM

- Centralización de la información
- Automatización de procesos
- Acceso desde distintos departamentos
- Gestión de usuarios y permisos
- Generación de informes
- Integración entre módulos
- Acceso web y en la nube
- Escalabilidad

---

# Ventajas de utilizar ERP y CRM

- Menos errores humanos
- Mayor productividad
- Mejor organización
- Información en tiempo real
- Automatización de tareas repetitivas
- Mejor atención al cliente
- Mejor toma de decisiones
- Mayor control empresarial

---

# Importancia para alumnos DAW

Como futuros desarrolladores web es importante conocer estos sistemas porque:

- Muchas empresas trabajan con ERP y CRM
- Existen APIs e integraciones
- Se desarrollan módulos personalizados
- Se automatizan procesos
- Se crean paneles web y aplicaciones conectadas
- Son muy demandados laboralmente

---

# SAP - Características principales

- Uno de los ERP más utilizados del mundo
- Orientado a grandes empresas
- Muy modular
- Gran capacidad de personalización
- Integración total entre departamentos
- Muy usado en multinacionales

---

# SAP - Puntos fuertes

## Puntos fuertes

- Muy pontente y completo
- Muy usado
- Mucho trabajo 


## Puntos débiles

- Curva de aprendizaje dura
- Lenguaje propio (ABAP)
- Muy caro

---

# Sage

## Características principales

- Muy usado en pequeñas y medianas empresas
- Orientado a contabilidad y gestión
- Fácil de implantar
- Interfaz relativamente sencilla
- Muy extendido en España

---

# Sage


## Puntos fuertes

- Fácil de usar
- Más económico
- Muy extendido
- Bueno para pymes


## Puntos débiles

- Menos potente que SAP
- Menor personalización
- Limitado para empresas muy grandes

- Funcionamiento cloud

---

# Dynamics 365


## Puntos fuertes

- Interfaz moderna
