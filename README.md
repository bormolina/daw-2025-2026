# Apuntes y código de DAW

Repositorio con apuntes, ejemplos de código y materiales de apoyo para los módulos de **1.º y 2.º de DAW** del **IES Francisco Javier de Burgos**.

## Contenido

En este repositorio se irán organizando distintos materiales utilizados durante el curso:

- ejemplos de código vistos en clase;
- comandos útiles de terminal;
- prácticas guiadas;
- ejercicios de repaso;
- pequeños proyectos de ejemplo;
- documentación complementaria.

## Objetivo

El objetivo del repositorio es servir como apoyo al trabajo de clase, permitiendo consultar ejemplos, repasar procedimientos y acceder de forma ordenada a materiales relacionados con los distintos módulos del ciclo.


## Uso del repositorio

El alumnado puede consultar este repositorio para:

- revisar ejemplos trabajados en clase;
- copiar comandos necesarios para algunas prácticas;
- descargar código base;
- comparar soluciones;
- repasar antes de pruebas o entregas.

## Aviso

Este repositorio es un material de apoyo. No sustituye a las explicaciones de clase, a las instrucciones concretas de cada práctica ni a las indicaciones dadas por el profesor.
